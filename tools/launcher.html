Git — Sync
Pull latest changes from GitHub
git pull origin claude/remove-exposed-api-key-ThrUa
Pull from main branch
git pull origin main
Push changes to GitHub
git push
Check status — see what's changed
git status
Stage all changes and commit
git add . && git commit -m "Update"
Setup
Create brand-kit/config.js with API key
echo "var GOOGLE_API_KEY = '...';" > brand-kit/config.js
Open project in VS Code
code .
Serve site locally (requires Python 3)
python3 -m http.server 8080
How it works: Clicking "Run" downloads a .terminal file. Double-click it in Finder or from your browser's download bar — macOS opens Terminal and runs the command automatically, starting from your project folder. No installation required.
